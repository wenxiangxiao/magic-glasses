# 🕶️ 神奇眼鏡 - 顏色疊疊樂

兒童顏色混合學習遊戲，透過疊加透明色片學習顏色混合原理。

## 🎮 遊戲模式

### 單人模式
- **🎮 單人闖關** - 10 關難度遞進
  - 第 1-2 關：單色 + 基本花紋
  - 第 3 關起：加入計時
  - 第 4 關起：雙色混合
  - 第 6 關起：新增黑色
  - 第 7 關起：新增白色
  - 第 8 關起：三色混合
- **🎯 自由練習** - 選擇難度自由玩

### 雙人對戰
- **⚡ 搶答模式** - 先答對得分，題目同步刷新
- **🎯 各自答題** - 各答各的，先答完全部題目獲勝

### 學習模式
- **📚 教學模式** - 互動式學習顏色混合原理

## 🎨 顏色系統

### 基本色
- 🔴 紅色
- 🟡 黃色
- 🔵 藍色
- ⚫ 黑色（加深）
- ⚪ 白色（加淺）

### 混合規則
| 混合 | 結果 |
|------|------|
| 紅 + 藍 | 紫色 |
| 紅 + 黃 | 橙色 |
| 黃 + 藍 | 綠色 |
| 紅 + 黑 | 深紅 |
| 紅 + 白 | 粉紅 |
| 黑 + 白 | 灰色 |

## ✨ 花紋圖案
- 點點
- 斜條紋
- 格子
- 鋸齒
- 圓圈

## 🚀 快速開始

### 純前端（單人模式）
直接用瀏覽器開啟 `index.html`

### 含多人對戰
```bash
# 安裝依賴
npm install

# 啟動伺服器
node server.js
```
然後訪問 `http://localhost:3000`

## 📦 檔案結構

```
magic-glasses/
├── index.html     # 遊戲主頁（純前端）
├── server.js      # 多人遊戲伺服器
```

## 🛠️ 技術架構

- **前端：** 純 HTML/CSS/JavaScript（無框架）
- **後端：** Node.js + Express（僅多人模式需要）
- **通訊：** Polling（500ms）
